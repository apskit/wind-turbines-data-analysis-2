# Wstępne przetwarzanie i analiza danych pochodzących z turbin wiatrowych

Celem pracy jest implementacja metod przetwarzania danych SCADA pochodzących z turbin wiatrowych. W ramach przetwarzania danych zaimplementowane zostaną wybrane metody czyszczenia danych, analizy parametrów statystycznych oraz proste metody wykrywania anomalii. Dane analizowane w ramach projektu będzie pochodzić z darmowych ogólnodostępnych źródeł.
